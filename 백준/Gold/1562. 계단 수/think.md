# 배운점
- 0에서부터 9까지 다 써야한다는 조건을 처음부터 고려하려고 해서 어려웠던 것
- 그냥 계단수를 만드는 경우의 수를 구하고, 그 중에 다 썼는지를 추리는 방식으로 접근하면 됨
- 계단수를 만드는 경우의 수는 그냥 완탐으로 구하면 됨. 길이가 L, 마지막 숫자가 R이라고 상태를 정의하면 구할 수 있음
- 그렇게 길이 L의 계단수를 구했을 때, 0부터 9까지 다 썼는지는 어떻게 알까? => 그것도 상태로 두기
- 오... 0에서 9까지를 visited로 하면 배열이 너무 커지는데? => 비트마스킹
