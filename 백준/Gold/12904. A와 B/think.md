# 생각의 흐름
1. 일단 완탐은 아님 2의 1000승임
2. 규칙을 좀 찾아볼까 패턴을 표현할 수 있음
3. 순행일 때와 역순일 떄의 패턴 표현 가능
4. S와 일치하는 문자열을 T에서 찾을 때마다 이 규칙을 만족하는 지 확인하면 될 듯
5. 안됨 자꾸...
6. 그래서 걍 답 봄
